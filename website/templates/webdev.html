This website was created with Flask as the main development tool. Nearly all functionality is coded in Python, with some of the visualizations being presented in Javascript. The login, registration, and user authentication is handled with helper Flask libraries to make the development process easier. The backend is a non-relational database hosted with MongoDB Atlas. All of the routes are constructed using Flask with the assistance of html templates. If you would like to inquire about a web design project this is my preferred architecture to work with.
I have also developed websites with React, focusing on the MERN stack. An example of my work on a React site can be found on my Github. The frontend is this repository. The backend is this repository. Please let me know if you would like to inquire about a web design project.
